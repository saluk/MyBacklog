?You will need to set a location for the game database (should pop up on initial run)
If you want to import games, you will need to enter some account information from the services, under file:options
To import games, use import:all, or if you want to only check a specific service, import:service
You can also add games manually, via add game (usually add game:none)
Select a game from the list to see its properties or run it
To run a game that's not from steam, you will have to set it's install_path from the game's options on the left to the exe you want to use to start the game (it can be an exe, a shortcut file, or a batch file)
When you stop playing the game, remember to click on stop playing in MyBacklog to stop the timer
